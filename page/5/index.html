Set

ES6提供的一种新的数据结构Set。和数组类似但是成员的值都是唯一不重复的，本身是一个构造函数，用来生成Set 数据结构。

const s = new Set（）;
[2,3,4,5,6,7,2,2].forEach(x=>s.add(x));
for (let i of s){
    console.log(i);
}
//2,3,4,5,6,7

通过add（）方法向Set结构加入成员，结果表明Set结构不会添加重复值。

chrome web worker

Posted on 2020-01-31

Chrome with Webworker

If I try to use web workers through a JavaScript file, Chrome throws an error -

Uncaught SecurityError: Failed to create a worker: script at '(path)/worker.js' cannot be accessed from origin 'null'.

The solution is to set up a python server in this path.

python -m http.server 8888

Then it works fine.
